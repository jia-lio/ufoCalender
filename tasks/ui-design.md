# UFO캐처 피규어 캘린더 - UI/UX 설계

## 디자인 방향

일본 아케이드 감성을 현대적으로 해석. 따뜻한 오프화이트 배경에 절제된 포인트 컬러.
피규어 이미지가 주인공이 되는 콘텐츠 중심 레이아웃.

## 3. 화면별 레이아웃

### 3-1. [캘린더 모달] — 메인 페이지 (/)

**데스크탑 (1024px+)**
```
┌──────────────────────────────────────────────────────────────┐
│  [로고] UFO캐처 피규어 캘린더                                │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─ 캘린더 헤더 ──────────────────────────────────────────┐  │
│  │     [← 이전달]     2026년 4월     [다음달 →]           │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                              │
│  ┌────┬────┬────┬────┬────┬────┬────┐                       │
│  │ 일 │ 월 │ 화 │ 수 │ 목 │ 금 │ 토 │  ← 요일 헤더         │
│  ├────┼────┼────┼────┼────┼────┼────┤                       │
│  │    │    │    │ 1  │ 2  │ 3  │ 4  │                       │
│  │    │    │    │[00시]    │    │    │  ← 시간대 태그        │
│  │    │    │    │[14시]    │    │    │                       │
│  ├────┼────┼────┼────┼────┼────┼────┤                       │
│  │ 5  │ 6  │ 7  │ 8  │ 9  │ 10 │ 11 │                       │
│  │[12시]    │    │    │    │    │    │                       │
│  ├────┼────┼────┼────┼────┼────┼────┤                       │
│  │ ...                              │                       │
│  └────┴────┴────┴────┴────┴────┴────┘                       │
└──────────────────────────────────────────────────────────────┘
```

**날짜 셀 디자인 상세**

피규어 없는 날:
```
┌────────┐
│ 15     │
│        │
└────────┘
```

피규어 있는 날 (시간대 태그만 표시):
```
┌────────┐
│ 1      │
│ [00시] │  ← 00시에 피규어가 있음
│ [14시] │  ← 14시에 피규어가 있음
└────────┘
```

같은 시간대에 여러 피규어가 있어도 시간대 1개만 표시:
```
┌────────┐
│ 5      │
│ [12시] │  ← 12시에 피규어 3개 있어도 [12시] 1개만
│ [18시] │
└────────┘
```

### 3-2. [날짜 모달] — 캘린더 날짜 클릭 시

[캘린더 모달]에서 날짜를 클릭하면 열림.
X를 누르면 [캘린더 모달]로 돌아감.

```
┌──────────────────────────────┐
│  4월 1일 (수)            ✕   │  ← ✕ 누르면 [캘린더 모달]로
├──────────────────────────────┤
│                              │
│  00:00  원피스 루피        >  │  ← 클릭하면 [상세 모달]
│  ─────────────────────────── │
│  00:00  나루토 사스케      >  │
│  ─────────────────────────── │
│  14:00  귀멸의 칼날 탄지로 >  │
│  ─────────────────────────── │
│  14:00  주술회전 고죠     >  │
│                              │
└──────────────────────────────┘
```

- 시간순 정렬
- 같은 시간의 피규어는 연속 표시
- 각 행을 클릭하면 [상세 모달] 열림

### 3-3. [상세 모달] — 피규어 클릭 시

[날짜 모달]에서 피규어를 클릭하면 열림.
X를 누르면 [날짜 모달]로 돌아감.

```
┌──────────────────────────────┐
│                          ✕   │  ← ✕ 누르면 [날짜 모달]로
├──────────────────────────────┤
│                              │
│  ┌────────────────────────┐  │
│  │                        │  │
│  │                        │  │
│  │      피규어 이미지      │  │
│  │        (큰 사이즈)      │  │
│  │                        │  │
│  │                        │  │
│  └────────────────────────┘  │
│                              │
│  ワンピース ルフィ            │  ← 일본어
│  원피스 루피                  │  ← 한국어
│  One Piece Luffy             │  ← 영어
│                              │
└──────────────────────────────┘
```

### 모달 네비게이션 요약

```
[캘린더 모달]  ←── ✕ ──  [날짜 모달]  ←── ✕ ──  [상세 모달]
  ← 이전달                                         (뒤로가기 없음)
  다음달 →                (뒤로가기 없음)
```

### 3-4. 모바일 뷰 (639px 이하)

캘린더 그리드 대신 **날짜별 목록 뷰**로 전환 (피규어가 있는 날짜만 표시):

```
┌──────────────────────────┐
│ UFO피규어캘린더            │
├──────────────────────────┤
│ [◀] 2026년 4월 [▶]       │
│                          │
│ ── 4월 1일 (수) ──────── │
│  [00시] [14시]           │  ← 시간대 태그, 탭하면 날짜 모달
│                          │
│ ── 4월 5일 (일) ──────── │
│  [12시]                  │
│                          │
│ ── 4월 12일 (일) ─────── │
│  [00시] [18시] [21시]    │
│                          │
└──────────────────────────┘
```

- 날짜/시간대 탭 시 동일한 날짜 모달 → 상세 모달 흐름

## 4. 컴포넌트 구조

| 컴포넌트 | 위치 | 설명 |
|----------|------|------|
| `CalendarView` | `components/calendar/` | 월간 캘린더 메인 (데스크탑 그리드 + 모바일 목록 전환) |
| `CalendarHeader` | `components/calendar/` | 년/월 네비게이션 |
| `CalendarCell` | `components/calendar/` | 일별 셀 (날짜 + 시간대 태그) |
| `TimeTag` | `components/calendar/` | 시간대 태그 뱃지 (예: `[00시]`) |
| `DateModal` | `components/calendar/` | 날짜 클릭 시 피규어 시간/이름 목록 모달 |
| `FigureModal` | `components/calendar/` | 피규어 클릭 시 이미지 + 이름(일/한/영) 상세 모달 |
| `MobileListView` | `components/mobile/` | 모바일 날짜별 목록 뷰 |

## 6. 인터랙션 패턴

| 인터랙션 | 효과 | 시간 |
|----------|------|------|
| 모달 열기 | `scale(0.95→1)` + `translateY(8px→0)` | 180ms ease-out |
| 월 전환 | 좌/우 20px slide + fade | 200ms |
| 피규어 카드 로드 | staggered fade-in | 0~80ms 딜레이 |
| 버튼 호버 | 배경색 변경 + `scale(1.02)` | 150ms |
| 피규어 카드 호버 | 밝기 증가 + 그림자 | 즉시 |

### 날짜 네비게이션
- 이전달/다음달 버튼
- 오늘로 돌아가기 버튼
- 년/월 직접 선택 (드롭다운)
